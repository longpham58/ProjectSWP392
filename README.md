# Internal Training Management System (ITMS)

## 1. Project Overview
Internal Training Management System (ITMS) is a web-based application that supports internal training operations in enterprises, including course management, employee enrollment, attendance tracking, assessment, feedback, and reporting.

## 2. Technologies
- Spring Boot (MVC)
- Spring Data JPA
- Thymeleaf
- Microsoft SQL Server
- Spring Security
- Docker & Docker Compose

## 3. Main Features
- Course and training management
- Employee enrollment and approval workflow
- Attendance tracking
- Quiz and assessment after training
- Feedback and evaluation
- Role-based dashboards
- Training reports and statistics

## 4. User Roles
- HR (Admin)
- Trainer
- Manager
- Employee

## 5. Deployment
The system is deployed using Docker with application and SQL Server containers.
